# EcoWorkConnect: An Integrated Digital Platform for Authentic Sustainability Service Management

## Abstract

**EcoWorkConnect** is a specialized digital marketplace designed to bridge the gap between certified professionals and end-users in the fragmented sustainability sector. Built on a scalable MERN stack, the platform ensures verified selection of experts for services like Solar Panel Installation, Energy Audit, Rainwater Harvesting, Bio-Gas Services, Waste Management, Water Conservation, EV Charging Support, and Green Maintenance. The system features Proximity-Based Filtering for hyper-local expert discovery, Smart Credential Validation, real-time service tracking, and a secure 'Pay After Service' model to enhance user trust. The architecture consists of responsive frontend with HTML5, CSS3, JavaScript (ES6+), robust Node.js and Express.js backend with RESTful APIs, and MongoDB database with Mongoose ODM. Key technical achievements include 15+ RESTful API endpoints, 3 normalized database collections, complete authentication system with OTP verification, and real-time booking functionality. This framework directly supports SDG 7, 11, and 12 by fostering clean energy adoption and building sustainable urban communities through verified, location-aware service management.

**Keywords:** MERN Stack, Sustainability Services, Digital Marketplace, Authentication Systems, Real-time Tracking
