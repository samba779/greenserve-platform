## Slide 2: Project Overview
---
**What We Built & Why**

**🎯 Problem:** Fragmented sustainability sector with no bridge between certified professionals and end-users
**💡 Solution:** Digital marketplace for authentic sustainability service management

**🔧 Technologies Used:**
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Maps:** Bhuvan (ISRO) Maps framework
- **Security:** Smart Credential Validation

**📱 Features:** Proximity-Based Filtering, Smart Credential Validation, Real-time API integration, Pay After Service

---

## Slide 3: System Architecture
---
**How We Built It**

**🏗️ Architecture:**
```
Frontend → RESTful APIs → MongoDB Database
                ↓
        National Portals (MNRE, CPCB, BEE)
                ↓
          Bhuvan Maps Framework
```

**🔐 Security Implementation:**
- Smart Credential Validation for official certifications
- Suryamitra ID authentication
- Tamper-proof expert selection
- Secure payment processing

**📊 Database Design:**
- **Users:** End-user profiles and preferences
- **Professionals:** Certified service providers
- **Services:** Solar, Rainwater, E-Waste categories
- **Bookings:** Real-time tracking and payments
